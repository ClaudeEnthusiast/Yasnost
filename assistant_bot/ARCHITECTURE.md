# Архитектура: Telegram Personal Assistant

## Стек

| Слой | Технология |
|---|---|
| Runtime | Node.js 20+ |
| Telegram | node-telegram-bot-api |
| БД | PostgreSQL (Unix socket, peer auth) |
| AI / LLM | Claude API (`claude-sonnet-4-6`) |
| Транскрипция голоса | OpenAI Whisper API (`whisper-1`) |
| Документы | `docx` npm package |
| Крон | `node-cron` |
| Конвертация аудио | `ffmpeg` (системный) |

> Почему два AI: Claude = интеллект (разбор намерений, генерация документов, отчёты).
> Whisper = транскрипция (Claude пока не принимает аудио напрямую).
> Можно заменить Whisper на бесплатный `faster-whisper` на самом VPS.

## Флоу: Голосовое сообщение

```
Пользователь → 🎤 voice.ogg
      ↓
Telegram → скачиваем файл
      ↓
ffmpeg → конвертируем в mp3
      ↓
OpenAI Whisper → транскрипция текста (рус/англ)
      ↓
Claude (intent router) → определяет тип:
  ├── "задача"      → handlers/tasks.js → в БД
  ├── "расход"      → handlers/budget.js → в БД
  ├── "документ"    → handlers/documents.js → генерирует .docx
  └── "вопрос"      → Claude отвечает текстом
      ↓
Ответ пользователю в Telegram
```

**Промпт для intent router:**
```
Пользователь сказал: "{text}"
Определи намерение (один из вариантов):
- task: хочет создать задачу/напоминание
- expense: упоминает расход/трату/заплатил
- document: хочет создать КП/NDA/договор
- question: задаёт вопрос или что-то другое

Ответь JSON: { "intent": "...", "extracted": { ... } }
```

## Флоу: Дейли-бриф (09:00)

```
node-cron → каждый день 09:00 MSK
      ↓
Загружаем из БД:
  - Задачи со статусом todo/in_progress
  - Задачи с due сегодня или просроченные
  - Расходы за вчера
      ↓
Claude → форматирует в читаемое сообщение
      ↓
bot.sendMessage(TELEGRAM_USER_ID, ...)
```

**Пример утреннего сообщения:**
```
☀️ Доброе утро! 2 июня 2026

📋 ЗАДАЧИ НА СЕГОДНЯ
🔴 [3] Подготовить КП для Иванова — просрочено на 1 день
🟡 [7] Встреча с партнёрами — срок сегодня

📊 ВЧЕРА ПОТРАЧЕНО
💸 3 500 ₽ в 2 транзакциях
  • Обед — 800 ₽
  • Такси — 2 700 ₽

✅ Хорошего дня!
```

## Флоу: Генерация документа

```
/кп Иванову, продаём склад 500м² в Химках, цена 15 млн
      ↓
Claude API (claude-sonnet-4-6):
  system: "Ты юрист и бизнес-консультант М.К ИНВЕСТ.
           Генерируй профессиональные документы на русском."
  user: описание сделки
      ↓
Claude возвращает структуру документа (JSON с секциями)
      ↓
services/docx.js → сборка .docx с форматированием
      ↓
bot.sendDocument(chatId, buffer, { filename: "КП_Иванов.docx" })
```

## Флоу: Недельный отчёт (воскресенье 20:00)

```
Собираем за 7 дней:
  - Выполненные задачи
  - Созданные задачи
  - Все расходы (сумма + разбивка по категориям)
  - Сгенерированные документы
      ↓
Claude → генерирует аналитический отчёт:
  - Что сделано
  - Что зависло (задачи без прогресса)
  - На что обратить внимание
  - Расходы vs предыдущая неделя
      ↓
Telegram: текстовый отчёт + .docx опционально
```

## Команды бота

### Задачи (этап 1 — уже сделан)
- `/задача [текст]` — добавить задачу
- `/задачи` — активные задачи
- `/готово [id]` — отметить выполненной
- `/бриф` — быстрая сводка

### Бюджет (этап 2)
- `/расход [сумма] [комментарий]` — записать расход
- `/бюджет` — расходы за текущий месяц
- `/категории` — расходы по категориям

### Документы (этап 3)
- `/кп [описание]` — сгенерировать КП
- `/нда [описание]` — сгенерировать NDA
- `/договор [описание]` — сгенерировать договор

### Голос (этап 2)
- 🎤 Голосовое сообщение → автоопределение намерения

### Автоматические
- Дейли-бриф 09:00 каждый день
- Недельный отчёт воскресенье 20:00
